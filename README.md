# 타입스크립트기반의 포켓몬 도감

[사이트](https://my-pokemon-app-d036d.firebaseapp.com/)로 이동하기

■ 패키지 모듈 설치
$npm i

■ 개발 서버 열기
$npm run dev

## package

- React
- Vite
- Typescript
- tailwindcss
- firebase (auth/hosting)
